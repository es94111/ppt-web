# FlowNote
## 讓團隊知識真正流動

產品提案 · 2026

---

# 我們正在解決什麼？

- 文件散落在聊天、Email 與個人硬碟
- 新成員平均需要兩週才能找到關鍵資訊
- 重複詢問與重複製作持續消耗團隊時間

> 問題不是缺少文件，而是文件無法在需要時被找到。

---

# 解決方案

FlowNote 是一個以工作流程為核心的團隊知識平台。

1. 自動整理會議與決策
2. 依專案建立知識脈絡
3. 使用自然語言搜尋答案
4. 標示來源、負責人與更新時間
