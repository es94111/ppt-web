# 客戶成長

本季新增 286 家付費客戶，其中：

- 47% 來自內容與自然搜尋
- 31% 來自合作夥伴推薦
- 22% 來自付費廣告與活動

**洞察：** 合作夥伴通路的成交率最高，應增加資源投入。
