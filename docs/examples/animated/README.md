# 動畫 Markdown 簡報

這些檔案使用 [Slidev](https://sli.dev/) 語法，支援：

- `transition` 投影片轉場
- `v-click`、`v-after` 逐步顯示
- `v-motion` 進場動畫
- 自訂 HTML、CSS 與關鍵影格動畫
- Mermaid 圖表

## 預覽

安裝 Slidev 後執行：

```bash
npx slidev "docs/examples/animated/01-Aurora-產品發表會.md" --open
```

其他範例只需替換檔名。

> 動畫只會在網頁播放模式呈現；匯出 PDF 時會保留最終靜態畫面。
